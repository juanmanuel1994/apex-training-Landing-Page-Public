APEX TRAINING — Landing Page
============================

Premium gym landing page. Pure HTML/CSS/JS, no dependencies, no npm.
Just open index.html in a browser and it works.


FILES
-----
index.html  — full structure and content
style.css   — styles, responsive layout, animations
script.js   — interactive gradient, effects, form


WHAT IT DOES
------------
- Animated canvas gradient that follows the mouse cursor
- Sticky navbar that changes on scroll, with mobile menu
- Hero section with massive typography and entrance animation
- 6 program cards with 3D tilt effect on hover
- Scrolling text marquee band
- 4 trainer cards with parallax effect on hover
- 3 membership plans (Starter / Performance / Elite)
- Contact form with success animation
- Fully responsive down to mobile


SECTIONS
--------
1. Hero         — giant headline, stats, call to action
2. Programs     — 6 disciplines with description and frequency
3. Trainers     — 4 coaches with role and certifications
4. Memberships  — pricing and features per plan
5. Contact      — info + functional form


CUSTOMIZING
-----------
- Colors: edit the :root block at the top of style.css
- Text content: edit directly in index.html
- Trainer photos: replace .trainer-placeholder with an <img> tag
- Form: connect the submit handler in script.js (section 7) to your backend or a service like Formspree
